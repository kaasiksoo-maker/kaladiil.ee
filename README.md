# Kaladiilid.ee v4 – AI agent architecture

React + Vite frontend, Cloudflare Pages Functions API ja D1 andmebaasi skeem.

## Cloudflare Pages settings

Framework preset: React (Vite)  
Build command: npm run build  
Build output directory: dist

## Mis on sees

- AI-agendi otsingu endpoint `/api/agent-search`
- Eesti poodide register: Jahipaun, Salmo, Vobla, WFish, JahiKala, BoBo, e-Kalastus, Kalastuskeskus
- andmebaasi skeem toodetele, pakkumistele, hinnaloole ja hinnateavitustele
- kasutajaliides, mis eristab rulle, ritvu, nööre, jige ja lante

## Järgmine samm

Päris hindade jaoks tuleb iga poe kohta lisada ametlik feed/API või lubatud adapter.
